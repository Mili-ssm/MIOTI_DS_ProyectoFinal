# MIOTI_DS_ProyectoFinal

Proyecto final del Master en Data Science de MIOTI


## Instalar UV

- **Windows**

    ```bash
        powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```

- **MacOS / Linux**

    ```bash
        curl -LsSf https://astral.sh/uv/install.sh | sh
    ```

- **Pip**

    ```bash
        pip install uv
    ```

## Instalar Dependencias

```bash
    uv sync
```
